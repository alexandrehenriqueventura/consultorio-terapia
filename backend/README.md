# Backend

Diretório do backend da aplicação de gerenciamento de consultório de terapia psicológica.

## Estrutura

Este diretório contém toda a lógica do servidor, APIs e serviços backend.

## Tecnologias

- Node.js
- Express/Fastify
- Base de dados (PostgreSQL/MySQL)

## Instalação

```bash
npm install
```

## Execução

```bash
npm start
```
