# Database

Diretório de scripts e configurações do banco de dados.

## Estrutura

Este diretório contém:
- Scripts de criação de schema
- Migrations
- Seeds para dados iniciais
- Backups e restore scripts

## Tecnologias

- PostgreSQL/MySQL
- Scripts SQL
- Migrations tools

## Uso

### Criar banco de dados
```bash
npm run db:create
```

### Executar migrations
```bash
npm run db:migrate
```

### Seed data
```bash
npm run db:seed
```
