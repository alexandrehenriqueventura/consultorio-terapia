# Frontend

Diretório do frontend da aplicação de gerenciamento de consultório de terapia psicológica.

## Estrutura

Este diretório contém toda a interface de usuário e componentes visuais da aplicação.

## Tecnologias

- React/Vue/Angular
- HTML5/CSS3
- JavaScript/TypeScript

## Instalação

```bash
npm install
```

## Execução

```bash
npm run dev
```
